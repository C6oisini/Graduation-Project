## 第 3 页：摘要

**建议标题：**  
研究摘要：在高维语义空间中实现隐私保护与效用保持的折中

**三栏文字：**

**研究动机：**  
在端云协同的视觉语言模型部署场景中，终端通常不直接上传原始图像或文本，而是上传中间层 embedding。然而，中间特征并非“无语义”的随机表示，其中仍保留对象类别、空间结构、文本语义等可被反演攻击利用的信息。因此，仅依赖拆分推理并不能充分消除隐私泄露风险。

**方法概述：**  
本文从高维表征的几何结构出发，将归一化 embedding 建模为单位超球面上的方向点，并引入 von Mises-Fisher 分布构造正交切平面扰动机制。该机制通过控制方向偏转实现度量隐私，同时保持原始向量范数不变，从而减少对 Transformer 后续注意力计算和归一化层输入分布的破坏。

**主要结论：**  
基于 Qwen3-VL-8B-Instruct 的实验表明，在 MS-COCO 与 VQA-v2 任务中，vMF 机制在强隐私预算下相较 Gaussian 与 Laplace 等加性噪声机制表现出更稳定的隐私-效用折中，能够降低特征反演攻击成功率，同时保留较高的视觉理解与文本生成能力。

- 因此，有必要构造一种与高维语义空间几何结构相协调的隐私扰动机制，使隐私保护从“坐标加噪”转向“几何约束下的语义扰动”。

## 第 9 页：前置定义 - 度量隐私与 vMF 分布

**建议标题：**  
度量隐私为连续语义空间中的不可区分性提供形式化描述

**左侧：度量隐私思想**  
标准差分隐私通常针对离散数据库查询，其相邻关系由记录替换或删除定义。对于连续 embedding 空间，更自然的方式是用距离函数刻画样本之间的相似程度。度量隐私将输出分布的可区分性与输入之间的距离绑定，使语义相近的输入在机制输出上具有更强不可区分性。

**中间：vMF 分布**  
von Mises-Fisher 分布是定义在单位超球面上的方向分布，其概率密度围绕均值方向集中。集中度参数控制采样方向与均值方向的偏离程度，适合描述归一化向量空间中的方向扰动。

**右侧：本文适配方式**  
本文将 embedding 的隐私扰动从欧氏坐标空间转移到球面方向空间，在保持模长的前提下改变方向，从而使扰动机制与 VLM 表征空间的语义结构更加一致。

## 第 11 页：核心算法 - vMF 正交切平面扰动

**建议标题：**  
vMF 扰动通过方向偏转实现隐私保护，并严格保持 embedding 范数

**算法描述：**
1. 对输入向量 `x` 进行模长与方向分解：`r = ||x||`，`μ = x / r`。
2. 采样标准高斯随机向量 `n`，并将其投影到 `μ` 的正交切平面：`n⊥ = n - (n·μ)μ`。
3. 使用隐私预算控制扰动尺度：`λ = β / ε`。
4. 生成未归一化扰动方向：`z = μ + λn⊥`。
5. 对 `z` 重新归一化得到扰动方向：`μ′ = z / ||z||`。
6. 恢复原始模长得到输出：`y = r · μ′`。

**关键性质：**
- 范数保持：`||y|| = ||x||`。
- 方向可控：ε 越小，方向偏转越大，隐私保护越强。
- 几何一致：扰动发生在单位超球面及其切空间中，符合归一化 embedding 的结构假设。

## 第 12 页：扰动参数 - 隐私预算与语义相似度

**建议标题：**  
隐私预算 ε 控制方向偏转强度，从而调节隐私与效用的平衡

**可放入 PPT 的文字：**
- 在 vMF 机制中，隐私预算并不直接对应坐标噪声方差，而是通过 `λ = β / ε` 控制球面方向偏转幅度。
- 当 ε 较小时，扰动方向与原方向夹角增大，攻击者更难从输出 embedding 中恢复原始语义，但模型效用也会相应下降。
- 当 ε 较大时，扰动方向更接近原方向，生成质量和问答准确率提升，但反演风险也随之增加。
- 因此，vMF 机制提供了一个以角度偏转为核心的隐私-效用调节方式，使扰动强度具有明确几何解释。

**建议数据：**
- vMF ε=0.1：cosine similarity = 0.337。
- vMF ε=0.5：cosine similarity = 0.692。
- vMF ε=1.0：cosine similarity = 0.848。

## 第 16 页：隐私性证明思路

**建议标题：**  
vMF 机制的隐私性可由球面输出分布的概率密度比进行刻画

**证明逻辑：**
- 首先，将输入 embedding 映射为单位方向向量，并以球面测地距离定义相邻输入之间的语义距离。
- 其次，利用 vMF 分布或其高维切平面近似描述输出方向的概率分布。
- 对任意两个输入方向 `μ` 与 `μ′`，比较机制输出同一方向 `z` 的概率密度比。
- 通过球面距离的三角不等式，可将该密度比上界控制在与 `d(μ, μ′)` 相关的指数形式中。
- 因此，机制满足近似度量隐私：输入越接近，输出分布越不可区分。

**页内公式建议：**
```text
d(μ, μ′) = arccos(<μ, μ′>)
Pr[M(μ)=z] / Pr[M(μ′)=z] ≤ exp(ε · d(μ, μ′))
```

## 第 17 页：效用性分析

**建议标题：**  
范数保持与角度可控共同解释 vMF 的效用优势

**1. 范数保持性**  
vMF 输出形式为 `y = r · μ′`，其中 `r = ||x||`，因此输出范数与输入范数严格一致。该性质避免了加性噪声在高维空间中常见的范数膨胀问题。

**2. 语义相似度可控**  
由于模长不变，扰动前后的余弦相似度主要由方向夹角决定。隐私预算 ε 实际上控制了语义方向偏转幅度，因此效用损失具有清晰几何解释。

**3. Attention 稳定性**  
Transformer 的注意力点积与 embedding 范数和夹角共同相关。vMF 保持范数后，扰动主要体现为可控角度变化，有助于维持后续注意力权重的相对稳定。

1. 输入图像与文本样本。

3. 对每个 embedding 执行范数分解与单位方向归一化。
4. 在正交切平面中生成噪声并按 ε 缩放。
5. 重投影到单位球面并恢复原始范数。
6. 将扰动特征输入云端语言模型完成生成或问答。
7. 同时计算任务效用指标、反演风险指标和几何稳定性指标。

**页面强调句：**  
该流程将算法机制、模型接入和攻击评估整合为可复现实验闭环。

## 第 20 页：实验结果 1 - 主任务效用

**建议标题：**  
在强隐私约束下，vMF 相比加性噪声保留更高任务效用

**实验设置：**
- 基础模型：Qwen3-VL-8B-Instruct。
- 数据集：MS-COCO 图像描述任务与 VQA-v2 视觉问答任务。
- 对比机制：Pixel-Gaussian、Embedding-Laplace、vMF-Ours。
- 隐私预算：ε = 0.1、0.2、0.5、1.0、2.0、5.0。

**关键结果：**
- 在 MS-COCO 上，当 ε=0.1 时，vMF-Ours 的 CIDEr 为 84.9，高于 Embedding-Laplace 的 75.1 和 Pixel-Gaussian 的 58.4。
- 在 VQA-v2 上，当 ε=0.1 时，vMF-Ours 的 Accuracy 为 62.0，高于 Embedding-Laplace 的 56.4 和 Pixel-Gaussian 的 45.7。
- 随着 ε 增大，各机制效用均逐步接近无隐私基线，但 vMF 在强隐私区域优势更明显。

**结论句：**  
结果表明，保持 embedding 几何结构能够显著缓解强扰动下的语义效用损失。

## 第 21 页：实验结果 2 - 隐私泄露风险

**建议标题：**  
vMF 在降低反演成功率的同时维持较高任务效用

**可放入 PPT 的文字：**
- 本文采用 SSIM 与 AttackSuccessRate 衡量特征反演风险。SSIM 越低，表示攻击者恢复出的图像结构与原图越不相似；ASR 越低，表示攻击成功概率越低。
- 在 MS-COCO 上，当 ε=0.1 时，vMF-Ours 的 ASR 为 0.161，低于 Embedding-Laplace 的 0.206 和 Pixel-Gaussian 的 0.231。
- 在 VQA-v2 上，当 ε=0.1 时，vMF-Ours 的 ASR 为 0.159，也低于两类加性噪声机制。
- 从隐私-效用曲线看，vMF 点位更接近“低攻击成功率 + 高任务效用”的理想区域，说明其并非单纯通过牺牲效用换取隐私。

**结论句：**  
vMF 的优势来自与 embedding 几何结构相匹配的扰动方式，而不仅是噪声强度差异。

## 第 24 页：理论分析 1 - 隐私性定理

**建议标题：**  
输出分布的概率密度比可由输入间球面距离上界控制

**可放入 PPT 的文字：**
- 设归一化输入方向为 `μ` 和 `μ′`，二者之间的语义距离由球面测地距离 `d(μ, μ′)` 描述。
- 对任意输出方向 `z`，vMF 机制诱导的概率密度与 `z` 相对于输入方向的角度相关。
- 通过比较 `M(μ)` 与 `M(μ′)` 在同一输出 `z` 上的概率密度，可得到与 `d(μ, μ′)` 成指数关系的上界。
- 该结果说明：语义距离越近的输入，其扰动输出分布越难区分，从而满足度量隐私的基本思想。

**页内公式建议：**
```text
d(μ, μ′) = arccos(<μ, μ′>)
Pr[M(μ)=z] / Pr[M(μ′)=z] ≤ exp(ε · d(μ, μ′))
```

## 第 25 页：理论分析 2 - 范数保持性

**建议标题：**  
严格范数保持为 Transformer 推理稳定性提供结构性保证

**可放入 PPT 的文字：**
- vMF 机制输出为 `y = r · μ′`，其中 `r = ||x||`，`μ′` 为单位方向向量。
- 因此 `||y|| = ||r · μ′|| = r = ||x||`，即扰动前后向量模长严格一致。
- 该性质与 Gaussian / Laplace 加性噪声形成对比：后者在高维空间中容易引入额外能量，导致范数均值上升。
- 对 Transformer 而言，范数稳定有助于维持 LayerNorm 输入分布和 attention 点积量级，减少因扰动导致的系统性推理偏移。

**建议数据：**
- vMF ε=0.1 / 0.5 / 1.0：norm_mean 均为 1.000。
- Laplace ε=0.1：norm_mean = 1.237。
- Gaussian ε=0.1：norm_mean = 1.204。

## 第 26 页：理论分析 3 - 语义相似度与点积偏差

**建议标题：**  
方向偏转使语义损失具有可解释的几何含义

**可放入 PPT 的文字：**
- 在范数保持条件下，扰动前后 embedding 的余弦相似度等价于原方向 `μ` 与扰动方向 `μ′` 的夹角余弦。
- 因此，ε 对模型效用的影响可以通过角度偏转解释：ε 较小时，方向偏转更大，隐私增强但语义相似度下降；ε 较大时，方向更接近原始表示，模型效用提升但隐私保护减弱。
- 对 attention 计算而言，点积由向量范数和夹角共同决定。由于 vMF 保持范数，点积变化主要来自方向夹角，因此扰动对 attention 排序的影响更可分析、更可调节。

**公式建议：**
```text
cos(x, y) = <x, y> / (||x||·||y||) = <μ, μ′>
```

## 第 29 页：研究总结

**建议标题：**  
本文从几何结构出发，为 VLM 中间特征隐私保护提供了新的实现路径

**工程贡献：**
- 基于 Qwen3-VL-8B-Instruct 实现可插拔 privacy wrapper，在视觉 projector 输出和文本 embedding 输出处注入隐私层。
- 支持多种机制统一对比，并记录角度偏差、范数比等运行时统计量。

**实验贡献：**
- 在 MS-COCO、VQA-v2 和 LLaVA 复现实验中验证 vMF 的隐私-效用优势。
- 实验结果表明，vMF 在强隐私区域能够降低反演攻击成功率，并保留更高视觉理解与文本生成能力。

- 面向自适应几何攻击开展鲁棒性评估。
